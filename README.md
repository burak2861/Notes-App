# Android-Notes-App-Kotlin
Kotlin (MVVM) Android not defteri uygulaması.

---
## Özellikler
* Not ekleme, düzenleme ve silme
* Notları tarihe göre sıralama
* Notları arama
* Detay ve liste geçişlerinde animasyon

---
## Kullanılan Teknolojiler
* MVVM
* Room Database
* Coroutine
* Dagger-Hilt
* ViewModel
* LiveData
* ViewBinding
* Clean Architecture
